# Brand MRI + 처방전 상품 상세 기획서

> **상품명**: Brand MRI (브랜드 MRI)
> **가격**: 50만원 (1회 풀패키지) → 월 80만원 (월간 관리 전환)
> **타겟**: 중견 뷰티 브랜드, 스킨케어 브랜드, 수출 브랜드
> **한 줄 정의**: "AI가 귀사를 어떻게 보는지 정밀 촬영하고, 처방하고, 치료 인프라까지 구축합니다"

---

## 1. 상품 정의

### 1.1 문제 정의: "진단만 받고 병원을 못 찾는" 한국 브랜드

```
현재 시장의 구조적 문제:

Step 1. ❌ 진단 도구 자체가 없음
  → "AI에서 우리 브랜드가 어떻게 나오는지 알 방법이 없어요"
  → Brand MRI Snapshot이 이걸 해결

Step 2. ❌ 진단 받아도 처방을 못 받음
  → "공백이 있다는 건 알겠는데, 어떤 콘텐츠를 써야 하죠?"
  → Q-Intelligence + Answer Gap이 우선순위를 알려줌

Step 3. ❌ 처방 받아도 치료할 인프라가 없음  ← ★ 가장 큰 병목
  → "콘텐츠를 썼는데, 어디에 올려야 AI가 읽죠?"
  → 자사 웹사이트: JSON-LD 없음, llms.txt 없음
  → 네이버 블로그: AI 크롤링 차단
  → 에이전시: AEO 전문 에이전시 자체가 없음

Brand MRI + 처방전 = Step 1~3을 하나의 상품으로 해결
"진단 → 처방 → 치료 인프라 구축 → 첫 치료까지"
```
