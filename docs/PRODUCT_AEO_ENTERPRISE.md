# AEO 엔터프라이즈 번들 상품 상세 기획서

> **상품명**: AEO Enterprise (가칭: "Brand MRI Enterprise")
> **가격대**: Starter 150만/월 → Pro 250만/월 → Elite 400만/월 → Strategic 별도 협의
> **타겟**: 중견~대기업 뷰티 브랜드, 글로벌 K-beauty 기업, 다수 제품 라인 보유 기업
> **한 줄 정의**: "AI 답변의 모든 전선을 자동으로 장악하는 풀스택 AEO 전환 시스템"

---

## 1. 상품 정의

### 1.1 엔터프라이즈 고객의 문제는 다르다

```
소상공인/소형 브랜드 (AEO 올인원 29~129만원)
→ "AI에서 존재감이 없다" = 존재감 만들기

중견 브랜드 (Brand MRI 50만→80만원)
→ "AI가 우리를 모른다" = 진단 + 첫 치료

대기업/엔터프라이즈 (이 상품)
→ "AI에서 우리가 1위여야 한다" = 지배적 지위 구축 + 유지
   • 제품 라인 20개+, 카테고리 10개+ 동시 관리
   • 경쟁사 5~10개 실시간 모니터링
   • 한국어 + 영어 + 중국어/일본어 멀티마켓
   • 내부 R&D/마케팅/경영진에게 주간 인텔리전스 공급
   • 에이전시 대행이 아닌 "내부 시스템"처럼 작동
   • AI 답변 위기 시 실시간 대응 (경쟁사가 인용을 뺏으면 즉시 알림)
```

### 2.3 각 Tier별 고객 페르소나

#### Starter (150만원/월) — "우리도 AEO를 시작해야 한다"

```
이름: 최민서 (38세, 중견 스킨케어 브랜드 마케팅 팀장)
회사: 연 매출 200억 스킨케어 브랜드, 올리브영 입점
팀: 마케팅 5명 (AEO 전담자 없음)
현재: SEO 에이전시에 월 200만원 지출 중, AEO는 안 하고 있음
고민: "경쟁사가 ChatGPT에서 추천되기 시작했는데, 우리는 안 나온다"

구매 동기:
• 150만원은 기존 SEO 에이전시 비용과 비슷 → 전환 or 병행 가능
• "진단부터 콘텐츠 생산, 웹사이트까지 한 번에" → 내부 인력 불필요
• 월 10편 콘텐츠 자동 → 팀장 혼자 관리 가능

기대 성과: 6개월 내 핵심 CQ 50개 중 15개에서 AI 인용 획득
```

#### Pro (250만원/월) — "AI에서 카테고리 TOP 3가 되겠다"

```
이름: 김지현 (42세, 중견 뷰티 기업 CMO)
회사: 연 매출 500억, 스킨케어/메이크업/바디 3개 라인
팀: 마케팅 15명 + 외부 에이전시 2사
현재: 구글 SEO는 잘하고 있으나, AI 검색에서는 경쟁사 대비 열세
고민: "3개 제품 라인 전체를 AI에서 커버해야 한다"

구매 동기:
• 3개 AI홈피 (라인별) + 150 CQ 모니터링 → 전 제품 라인 커버
• 5사 경쟁사 모니터링 + 주간 AI 브리핑 → CMO 주간 회의 소재
• Clone Transform 6채널 + 한→영/중 → 해외 수출 라인까지 커버
• 월 25편 콘텐츠 = 주 6편 이상 → 에이전시 대체 가능

기대 성과: 12개월 내 주력 카테고리 AI 인용률 TOP 3
```

#### Elite (400만원/월) — "AI 검색에서 압도적 1위를 유지하겠다"

```

구매 동기:

기대 성과: 주력 10개 카테고리 AI 인용 점유율 1위 유지
```
