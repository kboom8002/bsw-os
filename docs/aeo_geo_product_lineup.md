## Tier 0: 무료 AI 가시성 티저 진단 — "AI Quick Scan"

### 컨셉

> **"당신의 브랜드, AI 검색에서 보이나요?"**

잠재 고객이 브랜드명과 업종만 입력하면, **2분 이내**에 핵심 지표 3개를 무료로 제공합니다. 결과를 보여주되 "상세 분석은 유료로"라는 갈증(Information Gap)을 설계합니다.

### 고객에게 보여줄 것

| 항목 | 내용 | 공개 범위 |
|---|---|---|
| **AI 가시성 스코어** | 3~5개 핵심 질문에 대해 AI가 브랜드를 언급하는지 확인 | ✅ 점수만 공개 (세부 내역 잠금) |
| **경쟁사 대비 위치** | "업종 평균 대비 상/중/하" 한 줄 판정 | ✅ 등급만 공개 (순위 잠금) |
| **가장 위험한 GAP 1건** | "이 질문에서 경쟁사는 노출되지만 귀사는 누락됩니다" | ✅ 질문 1건만 공개 (나머지 N건 잠금) |
| **CTA** | "총 XX건의 기회가 발견되었습니다. 상세 분석 받기 →" | 유료 전환 유도 |

### 활용하는 시스템 기능 (최소한)

```
입력: 브랜드명, 업종
  ↓
[1] Fair Probe Templates에서 해당 업종의 핵심 질문 3~5개만 추출
  ↓
[2] Lightweight Metric Runner로 AI 응답 수집 (ChatGPT or Gemini 1개 엔진만)
  ↓
[3] 해당 브랜드 언급 여부만 확인 (AAS 약식 산출)
  ↓
[4] 경쟁사 1~2개와의 비교 (GAP 1건만 추출)
  ↓
출력: 스코어 + 등급 + GAP 1건 + CTA
```

| 시스템 모듈 | 사용 범위 | 제한 |
|---|---|---|
| `fair-probe-templates.ts` | 업종별 질문 5개만 | 전체 패널 사용 불가 |
| `lightweight-metric-runner.ts` | AI 엔진 1개만 | 멀티 엔진 비교 불가 |
| `per-layer-metrics.ts` | AAS만 산출 | BDR/CWR/OPP 미산출 |

### 비용 구조

| 항목 | 예상 비용 |
|---|---|
| AI API 호출 (5질문 × 1엔진) | ~₩50 (Gemini Flash 기준) |
| 서버 연산 | 무시할 수준 |
| **고객당 원가** | **~₩50** |

### 수익화 전략

- 무료이지만 **이메일 입력 필수** → 리드(Lead) 확보
- 결과 페이지 하단에 "유료 상세 분석" CTA → Tier 1 전환
- 진단 결과를 **PDF로 다운로드** 시 이메일 자동 발송 + 3일 후 후속 이메일

#### 🔬 Part 3 상세: Surface 역설계 보고서 (Tier 2 독점)

AI가 답변을 구성하는 방식을 **해부**하여, 왜 경쟁사는 노출되고 귀사는 누락되는지를 구조적으로 설명합니다.

| 섹션 | 내용 |
|---|---|
| **7-Dimension ERR 레이더 차트** | factoid/procedural/comparative/authority/schema/topical/geo 7차원 반영률 시각화 |
| **AEPI 스코어** | 업종별 가중치가 적용된 AI 검색 존재감 종합 점수 (0~100) |
| **YELLOW 처방전** | 사이트에 콘텐츠는 있지만 AI가 반영하지 않는 영역 → Schema 추가, E-E-A-T 신호 강화, 헤딩 개선 등 구체적 처방 |
| **RED 콘텐츠 갭** | 업종에서 AI가 다루지만 귀사 사이트에 없는 콘텐츠 → 신규 콘텐츠 생성 제안 |
| **WHITE 블루오션** | AI도 아직 다루지 않고, 어떤 브랜드도 선점하지 않은 질문 영역 → 선점 전략 |
| **Citation 맵** | AI가 경쟁사의 어떤 URL을 인용하는지 도메인별·질문유형별 매핑 |

#### 🔬 Part 4 상세: QVS 질문 포트폴리오 (Tier 2 독점)

질문을 단순히 나열하는 것이 아니라, 5차원 전략적 가치 평가 후 포트폴리오로 관리합니다.

| 섹션 | 내용 |
|---|---|
| **QVS 5-Dimension 스코어** | 각 질문의 volume_potential / ai_coverage_gap / brand_relevance / competition_intensity / conversion_proximity 점수 |
| **전략 분류 (Verdict)** | `prime_target`(즉시 공략) / `strategic_reserve`(축적) / `monitor`(관찰) / `deprioritize`(후순위) |
| **포트폴리오 건강도** | prime_target 비율, 평균 QVS 점수, 경쟁사 대비 포지션 |
| **🆕 예측 질문** | QVS Predictor가 발굴한 향후 3개월 내 부상 예측 질문 (계절·트렌드·규제 기반) |
| **질문 히트맵** | QVS 5차원 × 질문 매트릭스 색상 시각화 |

| 경쟁 서비스 | 한계 | BSW 차별점 |

| SEMrush / Ahrefs | 전통 SEO 지표만 (키워드 순위, 백링크) | **AI 검색 전용** 지표 (BDR, CWR, AEPI) |

| 일반 컨설팅사 | 수작업 분석, 고가, 느림 | **자동화된 파이프라인**으로 당일 납품 가능 |
| Perplexity / ChatGPT 직접 검색 | 한 번에 한 질문만, 체계적 분석 불가 | **업종별 프로브 패널** + **QVS 5차원 스코어링** |
