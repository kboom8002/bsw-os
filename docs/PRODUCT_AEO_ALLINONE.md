# AEO 올인원 상품 상세 기획서

> **상품명**: AEO 올인원 (가칭: "AI답변 올인원")
> **가격**: 월 29만원 (Basic) / 59만원 (Pro) / 129만원 (Premium)
> **타겟**: 소상공인, 소형 뷰티 브랜드, 1인 사업자
> **한 줄 정의**: "가입하면 AI가 추천하는 우리 가게가 완성됩니다"

---

## 1. 상품 정의

### 1.1 문제 정의: 한국 소상공인의 AEO 현실

```
소비자가 AI에게 묻는다: "강남역 근처 민감성 피부 관리 잘하는 곳?"
AI가 답한다: "A 피부관리실을 추천합니다. 이유는..."

Q: A 피부관리실은 어떻게 AI의 추천을 받았나?
A: 그 가게에 대한 정본 정보(JSON-LD, llms.txt, 구조화된 FAQ)가
   AI 크롤러가 읽을 수 있는 형태로 웹에 존재했기 때문.

Q: 한국 소상공인 대부분은?
A: ❌ 네이버 블로그 = AI 크롤링 차단
   ❌ 인스타그램 = AI 인용 불가
   ❌ 카카오맵 리뷰 = 구조화 데이터 없음
   ❌ 자체 웹사이트 = 없거나, JSON-LD 없음

→ AI 시대에 "디지털 존재감 제로" 상태
```

### 1.2 AEO 올인원이 해결하는 것

**"가입 → 업종 선택 → AI가 질문 발견 → 콘텐츠 자동 생산 → AEO 웹사이트에 발행 → 프로모션 연결"**

이 전체 과정을 고객이 직접 하는 것이 아니라, **시스템이 자동으로 수행**합니다.

### 1.3 구현 기술 스택 매핑

```
고객 여정              BSW-OS 모듈                    aihompy 모듈
─────────────────    ────────────────────────────    ─────────────────────────
가입 + 업종 선택  →   business-intake.ts            → Turnkey Engine (145KB)
                      (12KB, 업종별 인테이크)           Archetype System (12모듈)
                                                      Onboarding 위저드

AI 질문 발견     →   QEP (qis-local-qep.ts, 529줄)  → QIS Industry Templates
                      QVS (qis-local-qvs.ts, 179줄)    (22KB, 업종별 가중치)
                      Signal Collection (16모듈)

콘텐츠 자동 생산  →   AnswerAssetGenerator (342줄)    → Platform Writer Engine
                      AnswerMissionCompiler            Writer Hub (4단계 위저드)
                      HomepageComposer (202줄)         Clone Transform (20KB)
                      LlmTxtGenerator (101줄)

AEO 웹사이트 발행 →   JsonLdFactory (191줄)           → Storefront (멀티테넌트)
                      AnswerPageCompiler (213줄)        SEO Audit Engine (31KB)
                      hreflang-manager.ts               SEO Remediation (27KB)

프로모션 연결     →   (없음, aihompy에 위임)          → DealCard Engine (패키지)
                                                        Matching Engine
                                                        Gamification (Card Grade)

성과 측정        →    Observatory Probe               → Pulse Engine (8.9KB)
                      Signal Performance Tracker        Beauty Pulse (3모듈)
                      Citation Tracker
```
