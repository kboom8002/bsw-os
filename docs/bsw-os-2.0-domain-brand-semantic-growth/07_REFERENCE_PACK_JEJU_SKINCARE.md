## Golden Case

```text
비 오는 날 중문에서
유모차와 함께 이동하기 편하고
주차가 가능한 실내 중심 브런치 장소
```

## Failure Case

```text
제주 최고의 카페
근거 없음
실시간 영업 확인 없음
주차·접근성 단정
```

```text
건조하고 민감한 피부 사용자가
저자극 보습제품을 고를 때
성분·제형·사용법·주의사항을
근거수준별로 이해하는 가이드
```

## Failure Case

```text
이 성분이 피부염을 치료한다
사용자 후기만으로 효과를 일반화한다
임신 중 안전을 보장한다
```
