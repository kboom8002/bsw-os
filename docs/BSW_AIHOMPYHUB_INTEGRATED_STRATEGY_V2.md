# BSW-OS × AIHOMPYHUB 통합 진단-구축-성장 전략 고도화

> **Version:** 2.0 (2026-06-26)  
> **방법론:** Triple Diamond (발견→정의→개발→전달) + 혁신 사고(10x Thinking) + 로지컬 씽킹(MECE + 이슈 트리)  
> **범위:** BSW-OS 진단/평가 + AIHOMPYHUB 생태계 가동/운영의 완전 융합

---

## 0. Executive Summary

### 현행 전략(V1)의 한계

V1 Double Diamond 전략은 6가지 가치제안(VP-1~VP-6)을 **독립적**으로 정의했습니다. 
그러나 두 시스템의 실제 구현 감사 결과, **BSW-OS 28개 진단 컴포넌트**와 **AIHOMPYHUB의 GENESIS 5단계 파이프라인 + 3종 허브(업종/지역/테마) + Vortex DAO + 38개 디자인 테마**가 이미 고도로 구현되어 있어, VP들을 **선형 여정이 아닌 동시 가동형 플라이휠**로 재설계해야 합니다.

### V2 핵심 전환

```
V1: 진단(BSW) → 처방 → 구축(GENESIS) → 성장(QIS)      [선형 파이프라인]
V2: 진단⟷구축⟷성장이 동시 순환하는 통합 플라이휠        [순환 엔진]
```

```
  │   ├── industry: skincare_premium
  │   ├── archetype: "clean_scientific" (갭 분석 기반 자동 선정)
  │   └── surface_preset: L4_modern_clean (P75 통계 기반)
```

#### 🚀 Innovation 2: "3축 동시 입점" (Tri-Axis Auto-Enrollment)

**V1:** 허브 입점은 수동 관리  
**V2:** 테넌트 개설 시 자동으로 3축 모든 관련 허브에 입점

```

  상호: 구치미당 디저트카페
  주소: 제주시 한림읍
  업종: 카페/디저트
  핵심키워드: 한라봉타르트, 바다전망

자동 3축 매핑 결과:
  ├── Industry Hub: K-Food (k-food) ← 업종 자동 분류
  ├── Place Hub: Jeju (jeju) ← 주소→place_slug 매핑
  │   └── Sub-place: 한림 (hallim) ← 상세 지역 매핑
  ├── Vortex Hub: K-Experience (k-experience) ← 키워드 분석
  │   └── Theme: 제주감성카페 ← vortex_slug 자동 매핑
  └── Cross-Axis: "제주 디저트 카페" ← 3축 교차 키워드 자동 발견

  ⇒ 3개 Hub에 동시 입점
  ⇒ 3축 교차 콘텐츠 미션 자동 발행
  ⇒ 3개 Hub 리스트에 즉시 노출
```

**기술 구현 포인트:**

#### 🚀 Innovation 4: "AEPI-Driven Tenant FSM" (성과 기반 자동 등급 관리)

**V1:** 테넌트 상태 수동 관리  
**V2:** BSW AEPI 시계열 → 자동 등급 전이 + 처방전 발행

```
AEPI 기반 Tenant FSM 자동 전이 규칙:

  AEPI ≥ 80 (A등급) → PREMIUM 자동 승격
    └── 리더보드 상위 등재 + 허브 추천 슬롯 배치

  AEPI 60~79 (B등급) → ACTIVE 유지
    └── Growth 미션 지속 발행 + 월간 벤치마크 리포트

  AEPI 40~59 (C등급) → ACTIVE (경고)
    └── Quick Win 3개 자동 처방 + Polish 집중 실행

  AEPI < 40 (D등급) → DORMANT 위험
    └── 긴급 처방전 발행 + 관리자 알림 + 콘텐츠 미션 증량

  30일 연속 콘텐츠 미발행 → DORMANT 자동 전이
    └── 재활성화 미션 자동 발행 + 할인 쿠폰

  90일 DORMANT 유지 → CHURNED
    └── 재가입 유도 이메일 자동 발송
```

**기술 구현 포인트:**
- BSW `temporal-tracker.ts` AEPI 시계열 → AIHOMPYHUB `tenants.status` UPDATE
- BSW `strategy-generator.ts` Quick Win → AIHOMPYHUB `vortex_content_missions` INSERT
