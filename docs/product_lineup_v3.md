## Feature 0: 초기 AI홈피 구축 (온보딩)

### 네이버 플레이스 → AI홈피 자동 생성

```mermaid
sequenceDiagram
    participant OWNER as 사장님
    participant BSW as BSW-OS
    participant NAVER as 네이버 플레이스
    participant AI as AI 엔진
    participant HUB as 제주 AI Hub

    OWNER->>BSW: 네이버 플레이스 URL 입력
    BSW->>NAVER: NaverPlaceClient.fetchByUrlOrId()
    NAVER-->>BSW: 업체명, 주소, 전화, 영업시간, 카테고리
    BSW->>AI: inferIndustryType() + inferFacilities()
    AI-->>BSW: 업종 분류 + 시설 속성 추론
    BSW->>AI: extractContextConcepts() → 맥락 개념 추출
    BSW->>AI: findApplicableAttractors() → 어트랙터 매칭
    BSW->>AI: HomepageComposer.compose() → 섹션 조립
    BSW->>AI: generateSituationalFAQ() → FAQ 생성
    BSW->>AI: generateAltTexts() → 이미지 Alt Text
    BSW-->>OWNER: AI홈피 완성 (6 GNB, 24+ 에셋)
    BSW->>HUB: 퍼블리싱
```
