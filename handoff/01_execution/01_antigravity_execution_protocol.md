Version: v2.0-draft

---

## Prompting Protocol

For each batch, paste:

```text
1. batch README
2. batch master mission
3. definition of done
4. non-negotiables
5. relevant migration SQL
6. server action contract
7. UI route contract
8. test plan
9. review checklist
```

---

## Work Protocol

Antigravity should:

```text
implement smallest coherent slice
avoid overbuilding future modules
preserve folder structure
run tests/build where available
document gaps
do not hide failing tests
```

---

## Review Protocol

After each batch:

```text
inspect diff
run acceptance checklist
record gap report
decide patch/continue
```
