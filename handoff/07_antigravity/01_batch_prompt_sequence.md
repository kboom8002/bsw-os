## Prompt Sequence

For each batch, use:

```text
1. State batch goal.
2. Paste batch master mission.
3. Paste definition of done.
4. Paste non-negotiables.
5. Paste migration SQL.
6. Paste server action + UI + test contracts.
7. Ask Antigravity to implement minimal complete slice.
8. Ask it to run checks.
9. Ask it to create gap report.
```
